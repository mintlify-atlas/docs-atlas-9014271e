---
title: Register
description: Create a new user account.
---

`POST /api/auth/register`

<Note>No authentication is required for this endpoint.</Note>

## Request body

<ParamField body="username" type="string" required>
  Unique username, max 50 characters.
</ParamField>

<ParamField body="password" type="string" required>
  Account password, minimum 8 characters.
</ParamField>

<ParamField body="fullName" type="string" required>
  Full name of the user, max 100 characters.
</ParamField>

## Response

Returns the newly created user object. The `password` field is never included in the response.

<ResponseField name="id" type="string">
  Unique UUID identifier for the user.
</ResponseField>

<ResponseField name="username" type="string">
  The username chosen at registration.
</ResponseField>

<ResponseField name="fullName" type="string">
  Full name of the user.
</ResponseField>

<ResponseField name="createdAt" type="string">
  ISO 8601 timestamp of when the account was created.
</ResponseField>

## Example

<RequestExample>

```bash cURL
curl --request POST \
  --url http://localhost:3000/api/auth/register \
  --header 'Content-Type: application/json' \
  --data '{
    "username": "jhojanRop",
    "password": "12345678",
    "fullName": "Jhojan Ropero"
  }'
```

</RequestExample>

<ResponseExample>

```json 201
{
  "id": "a3f1c2d4-5e6b-7890-abcd-ef1234567890",
  "username": "jhojanRop",
  "fullName": "Jhojan Ropero",
  "createdAt": "2026-03-15T10:00:00.000Z"
}
```

</ResponseExample>

## Error responses

| Status | Description                                      |
| ------ | ------------------------------------------------ |
| `409`  | Username is already taken by an existing account |
| `400`  | Request body failed validation                   |
