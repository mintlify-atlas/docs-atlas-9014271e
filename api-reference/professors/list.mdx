---
title: List professors
description: Retrieve all professors belonging to the authenticated user.
---

`GET /api/professors`

<Note>
  This endpoint requires authentication. Include a valid Bearer token in the `Authorization` header.
</Note>

## Request

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer <access_token>`
</ParamField>

## Response

**200 OK** — Returns an array of professor objects.

<ResponseField name="id" type="string" required>
  Unique identifier of the professor (UUID).
</ResponseField>

<ResponseField name="fullName" type="string" required>
  Full name of the professor.
</ResponseField>

<ResponseField name="userId" type="string" required>
  ID of the authenticated user who owns this professor.
</ResponseField>

## Examples

<RequestExample>

```bash cURL
curl --request GET \
  --url http://localhost:3000/api/professors \
  --header 'Authorization: Bearer $ACCESS_TOKEN'
```

</RequestExample>

## Error responses

| Status | Description |
| ------ | ----------- |
| `401 Unauthorized` | No token provided or the token is invalid. |
