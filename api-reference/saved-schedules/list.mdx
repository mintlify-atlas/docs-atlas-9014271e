---
title: List saved schedules
description: Retrieve all saved schedules belonging to the authenticated user.
api: GET /api/saved-schedules
---

Returns all saved schedules for the authenticated user, ordered by creation date descending (most recent first).

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer <access_token>`
</ParamField>

## Response

Returns an array of `SavedSchedule` objects. Returns an empty array if no schedules have been saved.

<ResponseField name="id" type="string" required>
  UUID of the saved schedule.
</ResponseField>

<ResponseField name="name" type="string" required>
  Display name of the saved schedule.
</ResponseField>

<ResponseField name="createdAt" type="string" required>
  ISO 8601 timestamp of when the schedule was saved.
</ResponseField>

<ResponseField name="sections" type="object[]" required>
  The sections associated with this saved schedule.

  <Expandable title="section properties">
    <ResponseField name="sectionId" type="string" required>
      UUID of the section.
    </ResponseField>

    <ResponseField name="subject" type="string" required>
      Name of the subject.
    </ResponseField>

    <ResponseField name="professor" type="string | null" required>
      Full name of the professor, or `null` if unassigned.
    </ResponseField>

    <ResponseField name="startTime" type="string" required>
      Start time in `HH:MM` format (24-hour, UTC).
    </ResponseField>

    <ResponseField name="endTime" type="string" required>
      End time in `HH:MM` format (24-hour, UTC).
    </ResponseField>

    <ResponseField name="days" type="string[]" required>
      Days of the week this section meets.
    </ResponseField>
  </Expandable>
</ResponseField>

## Error responses

| Status | Description |
|--------|-------------|
| `401 Unauthorized` | Missing or invalid Bearer token. |

<RequestExample>

```bash cURL
curl --request GET \
  --url http://localhost:3000/api/saved-schedules \
  --header 'Authorization: Bearer '$ACCESS_TOKEN
```
